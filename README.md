# Pet-Clinic-App

**PetClinicApp** is a desktop application designed for pet owners and veterinary clinics. It allows easy management of customers, pets, services, and appointments from a single platform.  

Users log in first, then access Customer, Pet, Service, and Appointment forms through the **MenuStrip** on the main form. Each form supports adding, updating, deleting, and viewing data.  

---

## Features

- **Login Form**: User authentication.  
- **Main Form**: Access other forms via MenuStrip.  
- **Customer Form**: Manage customers.  
- **Pet Form**: Manage pets.  
- **Service Form**: Manage services.  
- **Appointment Form**: Manage appointments.  

---

## Technologies

- C# Windows Forms  
- **Database**: SQL Server  
- **Data Access**: ADO.NET
